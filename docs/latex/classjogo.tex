\doxysubsection{Detailed Description}
Classe base abstrata que representa um jogo genérico com um tabuleiro. 

Esta classe fornece a estrutura básica para um jogo que envolve um tabuleiro, com métodos para imprimir o tabuleiro e verificar se ele está cheio. Classes derivadas devem implementar os métodos virtuais.
